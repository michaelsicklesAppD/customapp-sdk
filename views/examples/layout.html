Layout Your Components

Best to start with newdash.html and build your dashboard.


	<body>

			<aside class="menu-sidebar">
				<div class="logo" style:"background:black">

				</div>
			</aside>

				<div class="container-fluid">

				</div>
			</div>

Within the page-container div, you will begin to add your components. Components and ChartComponents bind to div elements using the targetId property.

We use bootstrap grid layout to align components on the page see https://getbootstrap.com/docs/4.0/layout/grid

When working with Time Ranges, you need the spinnerDiv element for the progress animation.

Everything else is fully customizable

We have also made use of the open source Admin Bootstrap Theme see : https://github.com/ColorlibHQ/CoolAdmin

You can use font awesome icons see https://www.w3schools.com/icons/fontawesome_icons_intro.asp
