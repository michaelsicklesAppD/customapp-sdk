Within the page-wrapper div, you will begin to add your components. Components and ChartComponents bind to div elements using the targetId property.

We use bootstrap grid layout to align components on the page see https://getbootstrap.com/docs/4.0/layout/grid

When working with Time Ranges, you need the spinnerDiv element for the progress animation.

Everything else is fully customizable

We have also made use of the open source Admin2 Bootstrap Theme see : https://startbootstrap.com/template-overviews/sb-admin-2/

You can use font awesome icons see https://www.w3schools.com/icons/fontawesome_icons_intro.asp
