  • Within the page-container div, you will begin to add your components.
  • ChartComponents bind to div elements using the targetId property.
  • Bootstrap grid layout is used to align components (see : https://getbootstrap.com/docs/4.0/layout/grid)
  • Font Awesome icons are also used (see : https://www.w3schools.com/icons/fontawesome_icons_intro.asp)
