PieChartComponent

  • You can pass in your own data for you to visualize

	var pieData = [
			["Apple",1000],
			["Amazon",2000],
			["Cisco",3000],
			["AppDynamics",4000],
		]

    new PieChartComponent({
        targetId:"pieChart",
        title:"PieChart Component",
        data:pieData,
   }).draw(function(selection){
        alert(selection);
    })

PieChartComponent:

  • All chart components allow you to pass in an ADQL query instead of the data property.
  • If you do NOT provide any data or query values the component will generate mock performance data for you.
			
new PieChartComponent({
	targetId:"pieChartComponent",
	title:"PieChart Component"
	//query:"select customername,count(*) from transactions where application = 'test"
}).draw(function(selection){
	alert(selection);
})

Customizing Chart Options:

  • You can customize the chart by using the options property.
  • This will work for both Chart and ChartComponents.
			
	new PieChartComponent({
		targetId:"pieChartComponentCustomized",
		title:"PieChart Component Customized",
		options:{
			size:{width:"200",height:"200"},
			color:{pattern: ['#FF0000']},
			legend: {show: false}
		}
	}).draw(function(selection){
		alert(selection);
	})

Removing the outer layout:

  • You can remove the outer card by setting style:'none'
			
	new PieChartComponent({
			targetId:"pieChartNoLayout",
			title:"PieChart Component - No Card",
			style:'none'
    }).draw(function(selection){
        alert(selection);
    })
