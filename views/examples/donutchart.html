DonutChart Component:

  • Displays the data in donut chart
  • Wraps the chart in a default card layout
  • In this example we can pass in the data to be displayed
  • It defaults to showing the percentages

	var data = [
		["Customer Apple",1000],
		["Customer Amazon",2000],
		["Customer Cisco",3000],
		["Customer Facebook",4000],
	]
	
	new DonutChartComponent({
		targetId:"chart",
		data:data,
		title:"Customers"
	}).draw(function(selection){
		alert(selection);
	})

Donut Chart Component

  • All chart components allow you to pass in an ADQL query instead of the data property. If you pass in both, the query property would take precendence.
  • if you remove the data and query properties then the component will generate performance test data
			
	new DonutChartComponent({
		targetId:"chartComponent",
		title:"Donut Chart Component"
		//query:"select customername,count(*) from transactions where application = 'test"
	}).draw(function(selection){
		alert(selection);
	})

Customizing Chart Options

  • To customize the donut chart use the options attribute.
			
	new DonutChartComponent({
		targetId:"chartComponentCustomized",
		title:"All Red",
		options:{
			size:{width:"200",height:"200"},
			color:{pattern: ['#FF0000']},
			legend: {show: false}
		}
	}).draw(function(selection){
		alert(selection);
	})

Removing the card layout:

  • set style:none
  • This will remove the outer card layout
  • You can pass in your own style see the card css element.
			
	new DonutChartComponent({
			targetId:"chartNoLayout",
			style:'none',
			chartOptions:{
				size:{width:"200",height:"200"},
				color:{pattern: ['#FF0000']},
				legend: {show: false}
			}
		}).draw(function(selection){
			alert(selection);
		})
